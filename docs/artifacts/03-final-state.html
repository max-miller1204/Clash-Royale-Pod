Pipeline diagram, final form

VIDEO file (or HF replay frames)
  ↓ Timer.record("decode")
VideoFrameIterator → list[(idx, frame)]
  ↓ Timer.record("yolo")
YoloDetector.infer → list[Detection]
  ↓ Timer.record("tracker")
ByteTrack Tracker → list[Track] → list[CardPlay]
  ↓ Timer.record("hud")
HudReader.read → list[HudState] (HSV-mask princess-HP pixel sampling)
  ↓ Timer.record("features")
_assemble_replay → Replay
  ↓ Timer.record("ev")
analyze_replay
  ├── build_interactions(plays, hud, fps)        → list[Interaction]
  ├── [interaction_features(i) for i ...]        → feature rows
  ├── EvModel.predict(rows)                      → ev_predictions
  └── friendly_leak / enemy_leak / running_tempo
  ↓
_populate_blunders (CLI layer)
  └── detect_blunders(anchor_plays, preds, per_card_stats)
  ↓
output/analysis/<id>/
  ├── summary.json        (existing)
  ├── blunders.json       (Wave 3 — present when --model is supplied)
  ├── placements.png      (existing)
  ├── tempo.png           (existing)
  └── report.html         (Wave 4A — always present)
